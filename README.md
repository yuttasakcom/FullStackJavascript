## Install
```
$ git clone git@github.com:yuttasakcom/JSFullStack.git && cd JSFullStack && npm install
```

## Development
```
$ npm run dev
```

## Testing
```
$ npm run test

or

$ npm run cover
```

## Production
```
docker-compose up -d --build
```
